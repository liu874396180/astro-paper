# 文字相关项目汇总

开源笔记项目，开源文字阅读等项目汇总

## Table of contents

## anx-reader

- **GitHub 地址**: [https://github.com/Anxcye/anx-reader](https://github.com/Anxcye/anx-reader)
- **在线地址**: [https://anx.anxcye.com/](https://anx.anxcye.com/)
- **描述**: Anx Reader，一款专为爱书之人精心打造的电子书阅读器。它拥有强大的 AI 功能，支持多种电子书格式，让阅读更智能、更专注。凭借其现代化的界面设计，我们致力于为您带来纯粹的阅读乐趣。

## 微信 Markdown 编辑器

- **GitHub 地址**: [https://github.com/doocs/md](https://github.com/doocs/md)
- **在线地址**: [https://md.doocs.org/](https://md.doocs.org/)
- **描述**: Markdown 文档自动即时渲染为微信图文，让你不再为微信内容排版而发愁！只要你会基本的 Markdown 语法（现在有了 AI，你甚至不需要会 Markdown），就能做出一篇样式简洁而又美观大方的微信图文。

## memos

- **GitHub 地址**: [https://github.com/usememos/memos](https://github.com/usememos/memos)
- **在线地址**: [https://usememos.com/](https://usememos.com/)
- **描述**: 一款开源、可自行托管的笔记服务。你的想法，你的数据，你的控制权——无追踪，无广告，无订阅费。
- **预览图**: ![预览图](https://raw.githubusercontent.com/usememos/.github/refs/heads/main/assets/demo.png)

## NeuraPress

- **GitHub 地址**: [https://github.com/tianyaxiang/neurapress](https://github.com/tianyaxiang/neurapress)
- **在线地址**: [https://md.newkit.site/](https://md.newkit.site/)
- **描述**: NeuraPress 是一个现代化的 Markdown 编辑器，专注于提供优质的微信公众号排版体验。响应式设计，支持移动设备。搭配 DeepSeek 和微信公众号助手使用，碎片时间也能用手机发有排版的文章了。
- **预览图**: ![预览图](https://github.com/tianyaxiang/neurapress/raw/main/public/assets/img/neurapress-web-app.jpg)




## ebook-treasure-chest

- **github** : [https://github.com/jbiaojerry/ebook-treasure-chest](https://github.com/jbiaojerry/ebook-treasure-chest)

- **描述**: 欢迎来到电子书下载宝库，一个汇聚了各类电子书下载链接的地方。无论你是喜欢阅读经典文学、经管励志、终身学习、职场创业、技术手册还是其他类型的书籍，这里都能满足你的需求。 该库涵盖了帆书app(原樊登读书)、微信读书、京东读书、喜马拉雅等读书app的大部分电子书。
