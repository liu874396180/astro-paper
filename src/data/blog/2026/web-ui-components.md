## Uiverse
**在线地址**：[https://uiverse.io/](https://uiverse.io/)

**描述**：最大的开源用户界面库。社区构建的用户界面元素库。个人和商业用途均免费。

## Blocks
**GitHub地址**：[ephraim Duncan/blocks](https://github.com/ephraimduncan/blocks)

**在线地址**：[https://blocks.so/](https://blocks.so/)

**描述**：一个开源的 UI 模块库。基于 React、Tailwind 和 shadcn/ui 构建。

## UILayouts
**GitHub地址**：[ui-layouts/uilayouts](https://github.com/ui-layouts/uilayouts)

**在线地址**：[https://www.ui-layouts.com/](https://www.ui-layouts.com/)

**描述**：100 多个免费、美观、交互式的 React/Next.js 组件，采用 Tailwind CSS、Framer Motion、React Three Fiber、GSAP 等技术构建。

## 21st.dev
**在线地址**：[https://21st.dev/community/components](https://21st.dev/community/components)

**描述**：21st.dev 是一个专注于社区制作 UI 组件的平台。
